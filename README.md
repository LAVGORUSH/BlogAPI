BlogAPI(Backend - тестовое)
===============================

### Общее описание
На Spring boot требуется реализовать механиз блога по REST API.

### Описание сущностей
ВАЖНО!  Описание сущностей можно и нужно дополнять.
#### Категория:
- Заголовок
- Описание
- Фото
#### Статья:
- Заголовок
- Описание
- Фото
- Категория

### Запросы REST API:
1. Получение всех категория
2. Получение списка статей
3. Получение статьи
